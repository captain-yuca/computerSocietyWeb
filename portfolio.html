What's On Campus

Whats On Campus is an app designed to improve the communication between students and associations at universities. This app creates a platform where all organizations can advertise their workshops and events. The concept is to collect everything that is happening on campus in one platform.

3D Printer Companion App

The IEEE’s EMB Society is currently developing a 3D Printer as a project with the end goal of adapting the created system into a system that can print biological-related objects. This opens up the opportunity designing a mobile application that eases the usability of said system. This project will design, develop and implement an Android application in which users of the EMBS’s 3D printing system can utilize most of the printer's functionality remotely. This includes: the user can print, load CAD files, move the printer’s various axes remotely utilizing the app. The final output of the project will be an application in which the user can print remotely using CAD files from the app or SD card in the printer.
